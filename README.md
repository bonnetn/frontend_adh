2018 School project - ADH frontend
==================================

This is an application that was developped by Occuria and I. We had to
develo an Angular App. We chose to make something useful for our ISP
association so we made an app to handle the memberships and the devices of the
people.


Pour installer:
> Installation de NodeJS (voir le site). (pas la version 10, celle LTS en dessous)
> npm install --unsafe-perms -g @angular/cli@^1.7.4
> npm install

NOTE: La version 1.7.4 de angular-cli est la derniÃre version pour angular5

Pour lancer
> npm start
